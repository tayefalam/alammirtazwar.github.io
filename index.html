Key Project — Case Study

Car Damage Detection through Optimization of VGG16 CNN

📌 Problem

Existing CNN models rely heavily on large datasets and fail to detect subtle damages (scratches, dents) effectively when trained on limited or non-diverse data. Real-world automated damage assessment requires robust detection with minimal annotation cost.

⚙️ Approach

Leveraged transfer learning with VGG16 as base, built custom preprocessing pipeline using OpenCV (CLAHE, morphological ops). Applied targeted data augmentation (random erasing, cutmix, synthetic reflections) to improve generalization. Fine-tuned top layers with progressive unfreezing and cyclical LR scheduling.

✅ Solution

Improved detection accuracy for small, reflective damages by ≈14% mAP over baseline. Reduced overfitting significantly through regularization, dropout tuning, and class-weighted loss. Delivered a lightweight inference pipeline suitable for edge deployment.

📈 Impact

More stable validation performance with limited dataset; better real-world usability for automated damage detection systems. Demonstrates deep expertise in both CNN optimization and building end-to-end ML solutions under resource constraints.
