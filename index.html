Key Project — Case Study

Car Damage Detection · VGG16 Optimization

📌 Problem

CNNs fail on subtle damages (scratches/dents) with limited non-diverse data → significant accuracy gap in automated claims.

⚙️ Approach

Transfer learning (VGG16) + OpenCV preprocessing + targeted augmentation + cyclic LR scheduling.

✅ Solution

14.2% mAP improvement for reflective damages. 31% reduction in false negatives. Inference @ 45ms on edge.

📈 Impact

Production-ready pipeline with stable validation. Deployed in automated damage assessment prototype.
