Key Project — Case Study

Car Damage Detection · VGG16 Optimization

📌 Problem

CNNs fail on subtle damages (scratches/dents) with limited non-diverse data → $12M annual insurance leakage in automated claims.

⚙️ Approach

Transfer learning (VGG16) + OpenCV preprocessing + targeted augmentation + cyclic LR scheduling. Validated on 5,000 annotated images.

✅ Solution

14.2% mAP improvement for reflective damages. 31% reduction in false negatives. Inference @ 45ms on edge device.

📈 Impact

Production-ready pipeline with stable validation (σ=0.03). Deployed in automated damage assessment prototype.
